# PlaywrightZadanko


## Commands

- instal dependencies
  `npm i`
  - instal playwright
  `npx playwright install`
- run tests without browser GUI:  
  `npx playwright test`
- run test with browser GUI:  
  `npx playwright test --headed`
- viewing report  
  `npx playwright show-report`
